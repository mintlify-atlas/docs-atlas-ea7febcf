Get started with the BB Theme by following these essential guides. Whether you're installing the theme for the first time or learning to customize it, this section will help you get up and running quickly.

## Installation and setup

<Steps>
  <Step title="Install the BB Theme">
    Download and install both the parent theme and child theme from your Beaver Builder account.
    
    [View installation guide](/bb-theme/getting-started/install)
  </Step>
  
  <Step title="Access the Customizer">
    Learn how to open the WordPress Customizer to configure your theme settings.
    
    [View Customizer overview](/bb-theme/getting-started/customizer-overview)
  </Step>
  
  <Step title="Configure your settings">
    Customize your theme's appearance using the various Customizer tabs.
    
    [Explore Customizer settings](/bb-theme/customizer-settings)
  </Step>
</Steps>

## System requirements

Before installing the BB Theme, make sure your WordPress installation meets the [system requirements](/beaver-builder/getting-started/system-requirements) for viewing and building with the Beaver Builder plugin and theme.

## Next steps

Once you've installed the BB Theme, explore the [Customizer settings](/bb-theme/customizer-settings) to learn about all available customization options, or dive into the [defaults](/bb-theme/defaults) to understand how layouts, content, and styles work together.
