## What is the BB Theme?

The Beaver Builder theme, also known as the BB Theme, is a WordPress theme specifically designed to work seamlessly with the Beaver Builder plugin. The BB Theme provides a solid foundation for building beautiful and responsive websites. It offers a range of customizable options, including layout settings, typography, color schemes, and more.

By combining the power of the BB Theme with the Beaver Builder plugin, users can enjoy a streamlined website building experience.

## Key features

### Highly customizable

All BB Theme customization is done through the WordPress Customizer, giving you access to:

- Color schemes and accent colors
- Default fonts for headings and text
- Site header and footer layout and content
- Menus and widgets
- Custom global code for your site

### Multiple layout options

Choose from various header layouts including Nav Right, Nav Left, Nav Centered, Nav Bottom, and vertical navigation options. Control whether your site uses a full-width or boxed layout.

### Built for Beaver Builder

While not required, the BB Theme is optimized to work perfectly with the Beaver Builder plugin, providing a seamless website building experience.

### Clean and semantic code

The BB Theme is built with clean, semantic code that follows WordPress best practices, making it easy to customize and extend.
