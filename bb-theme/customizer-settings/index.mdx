The BB Theme provides extensive customization options through the WordPress Customizer. All theme settings are organized into intuitive tabs that let you control every aspect of your site's appearance.

## Overview

Access the Customizer by going to **Appearance > Customize** in your WordPress admin panel. The BB Theme adds several powerful tabs to control your site's design:

## General tab

The General tab contains fundamental settings for your entire site:

- **Layout** - Choose full-width or boxed layout, set content width, configure breakpoints
- **Background** - Set background color or image for the browser window
- **Accent Color** - Define colors for links, buttons, and interactive elements
- **Headings** - Set default styles for H1-H6 headings with responsive font sizes
- **Text** - Configure default text styles for body copy
- **Buttons** - Set global button styles used throughout your site
- **Social Links** - Configure social media icons and links

[View complete General tab documentation](/bb-theme/customizer-settings/general)

## Header tab

The Header tab controls everything in your site's header area:

- **Top Bar Layout** - Optional top bar with custom content or social icons
- **Top Bar Style** - Colors and styling for the top bar
- **Header Layout** - Choose from multiple header layouts (Nav Right, Nav Left, Nav Centered, Nav Bottom, Vertical)
- **Header Style** - Background colors, images, and text colors
- **Header Logo** - Upload logo images or use text, configure sizes and colors
- **Nav Layout** - Navigation menu spacing, search icon, mobile toggle options
- **Nav Style** - Navigation menu colors and typography

[View complete Header tab documentation](/bb-theme/customizer-settings/header)

## Content tab

The Content tab manages settings for your content area:

- **Content Background** - Set background color or image for content area
- **Blog Layout** - Sidebar position, post meta display options
- **Archive Layout** - Control how post lists are displayed
- **Post Layout** - Featured image, categories, tags, author box settings
- **WooCommerce Layout** - Sidebar position, columns, product gallery options
- **Lightbox** - Enable or disable lightbox for images

[View complete Content tab documentation](/bb-theme/customizer-settings/content)

## Footer tab

The Footer tab controls your site's footer area:

- **Footer Widgets Layout** - Enable footer widget areas (up to 4 columns)
- **Footer Widgets Style** - Background and text colors for widget area
- **Footer Layout** - Main footer with up to 2 columns of content
- **Footer Style** - Background and text colors for main footer
- **Footer Parallax** - Enable parallax scroll effect for footer

[View complete Footer tab documentation](/bb-theme/customizer-settings/footer)

## Additional tabs

### Presets

Quickly apply pre-designed color schemes and layouts. Choose from options like Default, Classic, Modern, Bold, Stripe, Deluxe, Premier, Dusk, and Midnight.

<Warning>
Changing a preset can overwrite your existing Customizer settings. Use presets as a starting point before making detailed customizations.
</Warning>

### Code

Add custom CSS and JavaScript code that applies globally across your site.

### Settings

Configure WordPress site identity (site title, tagline, site icon) and choose whether your home page displays latest posts or a static page.

### Menus

Create and manage navigation menus and assign them to menu locations.

### Widgets

Configure widgets for your sidebar and footer widget areas.

## Tips for using the Customizer

<Tip>
Use the device preview icons at the bottom of the Customizer panel to check how your site looks on desktop, tablet, and mobile devices.
</Tip>

<Tip>
Many settings support responsive configuration, allowing you to set different values for different screen sizes.
</Tip>

<Tip>
Always click **Publish** to save your changes. If you close the Customizer without publishing, your changes will be lost.
</Tip>

## Export and import settings

You can export your Customizer settings from one theme and import them into another. This is especially useful when:

- Moving from the parent theme to a child theme
- Migrating settings to a new site
- Creating a backup of your configuration

Learn more about [exporting and importing Customizer settings](/bb-theme/management-migration/export-or-import-customizer-settings).
