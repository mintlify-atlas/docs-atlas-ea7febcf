## Download

Download the Beaver Themer plugin zip from the [My Account](https://www.wpbeaverbuilder.com/my-account/) page. The download file is named `bb-theme-builder.zip`.

## Install

<Steps>
  <Step title="Navigate to the plugins page">
    From your WordPress Admin Dashboard, go to **Plugins > Add New**.
  </Step>
  
  <Step title="Upload the plugin">
    Click the **Upload plugin** button at the top of the screen.
  </Step>
  
  <Step title="Select the file">
    Click **Browse** and select the downloaded zip file (`bb-theme-builder.zip`) on your local system.
  </Step>
  
  <Step title="Install the plugin">
    After you have selected the file, click on the **Install Now** button.
  </Step>
  
  <Step title="Activate">
    Once the installation is complete, click on the **Activate** button.
  </Step>
</Steps>

<Tip>
You can also install Beaver Themer from the Beaver Builder settings page by going to the License tab and clicking the Install link in the Available Downloads section.
</Tip>

## Activation

If you activate the Beaver Themer plugin without having the Beaver Builder plugin installed, you'll see a warning. Additionally, Beaver Themer is not compatible with the [Lite version](https://wordpress.org/plugins/beaver-builder-lite-version/) of the Beaver Builder plugin. If both the Lite version and Beaver Themer are installed, the following message will appear at the top of the WordPress Plugins page:

<Warning>
Beaver Themer is not compatible with the lite version of Beaver Builder. Please purchase a premium version before continuing.
</Warning>
