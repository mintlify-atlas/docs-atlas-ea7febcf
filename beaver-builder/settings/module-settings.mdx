---
title: Module settings
description: Configure which modules are available in your Beaver Builder layouts.
---

The Module Settings tab in Beaver Builder allows you to enable or disable specific modules, controlling which modules appear in the Content Panel when building layouts.

## Enable/Disable modules

You can enable or disable individual modules based on your needs:

- **Enabled modules** will appear in the Content Panel's Modules tab when building layouts
- **Disabled modules** will be hidden from the Content Panel but existing instances in your layouts will remain functional

<Info>
Disabling a module only prevents it from being added to new layouts. Existing modules in published pages will continue to work normally.
</Info>

## Use cases

Module settings are useful for:

- **Simplifying the interface** for clients by hiding modules they don't need
- **Improving performance** by reducing the number of modules loaded
- **Maintaining consistency** by only allowing specific modules across a site
- **Creating custom workflows** for different user roles or site types

## Access module settings

<Steps>
  <Step title="Navigate to settings">
    Go to **Settings > Beaver Builder** in your WordPress admin.
  </Step>
  <Step title="Open Modules tab">
    Click the **Modules** tab.
  </Step>
  <Step title="Enable/Disable modules">
    Check or uncheck modules to enable or disable them.
  </Step>
  <Step title="Save changes">
    Click **Save Module Settings** to apply your changes.
  </Step>
</Steps>

<Tip>
If you're using Beaver Builder to build sites for clients, consider disabling advanced modules to create a simpler, more focused editing experience.
</Tip>
