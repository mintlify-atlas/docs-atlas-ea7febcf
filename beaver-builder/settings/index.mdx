## Access settings

<Steps>
  <Step title="Access WordPress admin">
    Access your site's WordPress Admin Dashboard.
  </Step>
  <Step title="Navigate to settings">
    Navigate to **Settings > Beaver Builder**.
  </Step>
</Steps>

<Info>
The Unlimited license of Beaver Builder allows white labeling to display any other name you choose and a custom logo image. Therefore, the label (Beaver Builder) on the **WordPress Admin Dashboard > Settings > Beaver Builder** may be different.
</Info>

## Network-wide settings (Unlimited license only)

The Network-wide settings are available on the WordPress Multisite installation main site when accessed as the **Network Admin** and allow you to configure Beaver Builder for all sub-sites on the network.

<Tip>
You can, for instance, enter your license key on the Network-wide settings and the license key will be used on all sub-sites, allowing the sub-site administrator to update Beaver Builder.
</Tip>

## Override Network-wide settings (Unlimited license only)

Users with administrator privileges can override Network-wide settings for their sub-sites.

<Steps>
  <Step title="Login to sub-site">
    Login to your sub-site WordPress Admin Dashboard.
  </Step>
  <Step title="Navigate to settings">
    Navigate to **Settings > Beaver Builder**.
  </Step>
  <Step title="Check override option">
    Check the **Override Network Settings?** option and save changes. If you want to override the network settings, you must check the **Override Network Settings** box for every Beaver Builder tab.
  </Step>
</Steps>

<Info>
The Welcome, License, Branding, and Help Button tabs are not available for sub-site administrators.
</Info>
