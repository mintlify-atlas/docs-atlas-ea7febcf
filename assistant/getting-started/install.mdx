---
title: Install the Assistant plugin
description: Learn how to install the Assistant plugin on your WordPress site using three different methods
---

There are three methods for installing the Assistant plugin, which are outlined below.

## Install from Assistant Pro website

<Steps>
  <Step title="Download the plugin">
    Download the Assistant Plugin from your [Assistant.Pro Cloud](https://assistant.pro/) dashboard. Click the **three dots** to toggle the dropdown menu and select **Download Plugin**.
  </Step>
  
  <Step title="Go to WordPress admin">
    From your WordPress Admin Panel, go to **Plugins > Add New** and click the **Upload plugin** button at the top of the screen.
  </Step>
  
  <Step title="Upload the plugin">
    Click **Browse** and select the downloaded zip file on your local system or just drag the file on top of the **Browse** button.
  </Step>
  
  <Step title="Install">
    Click **Install Now**.
  </Step>
</Steps>

![Download from Assistant pro website](/img/assistant/getting-started--installation--1.jpg)

## Install from WordPress.org

<Steps>
  <Step title="Download from WordPress.org">
    Download the Assistant Plugin from the [Wordpress.org](https://wordpress.org/plugins/assistant/) plugin repository.
  </Step>
  
  <Step title="Go to WordPress admin">
    Once downloaded, visit your website's WordPress Admin Panel, go to **Plugins > Add New** and click the **Upload plugin** button at the top of the screen.
  </Step>
  
  <Step title="Upload the plugin">
    Click **Browse** and select the downloaded zip file on your local system or just drag the file on top of the **Browse** button.
  </Step>
  
  <Step title="Install">
    Click **Install Now**.
  </Step>
</Steps>

![Download from Wordpress.org plugin repository](/img/assistant/getting-started--installation--2.jpg)

## Install from your WordPress site admin dashboard

To install the Assistant Plugin from your site's WordPress admin dashboard:

<Steps>
  <Step title="Navigate to Plugins">
    Navigate to **Plugins > Add New**.
  </Step>
  
  <Step title="Search for Assistant">
    In the Search box, type **"Assistant"**. You'll see the Assistant logo (yellow pencil icon), **Assistant - Every Day Productivity Apps** plugin name, description, and **by the Beaver Builder Team** in the search results.
  </Step>
  
  <Step title="Install">
    Click **Install Now**.
  </Step>
</Steps>

![Download from WordPress Admin Dashboard](/img/assistant/getting-started--installation--3.jpg)

## WordPress Multisite

You can install and use the Assistant plugin on [WordPress Multisite installations](https://wordpress.org/support/article/create-a-network/), either network-wide or on a per subsite basis.
